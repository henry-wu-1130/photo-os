# darktable Export Presets

This folder documents the darktable export preset settings.

darktable export presets are stored in its internal database, not as standalone files.
Use the settings below to recreate them on a new machine.

---

## Preset: photo-os web

| Setting | Value |
|---------|-------|
| Format | JPEG |
| Quality | 90% |
| Max width | 1080 px |
| Max height | 1080 px |
| Scale | Long edge |
| Color profile | sRGB |
| Filename | `$(FILE_NAME)_web.jpg` |
| Output path | `~/Photography/Export/[session]/web/` |

---
