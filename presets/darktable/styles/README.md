# darktable Styles

Place exported `.dtstyle` files here.

## Naming Convention

```
photo-os-base-v1.dtstyle          ← baseline for all images
photo-os-look-warm-v1.dtstyle     ← warm color grade
photo-os-look-film-v1.dtstyle     ← filmic B&W
photo-os-instagram-v1.dtstyle     ← Instagram crop + look
```

## How to Import into darktable

1. Open darktable.
2. `Settings → Styles → Import`.
3. Select the `.dtstyle` file from this folder.

## How to Export from darktable

1. `Settings → Styles → [select style] → Export`.
2. Save to this folder.
